When product is added to cart through Cart Notify plugin, popup notification will show on screen with product information.

=== Description ===

Cart Notify plugin gives you a very small and awesome feature that very attractively informs your user with a popup with beautiful animation that your product is included in the card.


== Features ==
* Popup Notification (when products are added to cart)
* Popup Purchase Notification (When the product order is completed)

== Up Coming Features ==
* Popup Notification designs
* Popup Notification location changing option
* Fake purchase Notification (After a certain period of time)

== Installation ==

1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. Alternatively, you can go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'Cart Notify'.
4. Install and activate the plugin.
5. That's it. No other settings is necessary. Now you will see The Notification when products are added to cart.

== If it's not working ==

- Step 1. Check if you Plugin is Activated.
- Step 2. Check each of your plugins has download button.
